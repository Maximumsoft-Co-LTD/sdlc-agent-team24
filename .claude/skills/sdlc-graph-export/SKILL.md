# SDLC Graph Export

## Steps
1. Run: `node scripts/sdlc-graph.js`.
2. Print the counts: nodes, edges by type.
3. Print the path to `docs/sdlc/_graph.mmd` so the user can view the Mermaid in any markdown viewer.
