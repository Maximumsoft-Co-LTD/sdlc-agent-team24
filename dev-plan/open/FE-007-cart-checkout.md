## งานที่ต้องทำ

### Cart Icon / Badge (ทุกหน้า)
- [ ] แสดงจำนวนสินค้าในตะกร้า (badge บน icon)
- [ ] อัปเดต real-time เมื่อเพิ่ม/ลบ

### ปุ่ม "เพิ่มในตะกร้า" (หน้า Book Detail)
- [ ] เฉพาะ type='buy' (ซื้อขาด) — ปุ่มเช่ายังใช้ flow เดิม POST /orders
- [ ] ถ้าอยู่ในตะกร้าแล้ว → เปลี่ยนเป็นปุ่ม "ดูตะกร้า"
- [ ] ถ้ามีสิทธิ์อยู่แล้ว → ปุ่ม "อ่าน" (ไม่มีตะกร้า)

### `/cart` — หน้าตะกร้า
- [ ] รายการหนังสือในตะกร้า + ราคาต่อเล่ม + ยอดรวม
- [ ] ปุ่มลบแต่ละรายการ
- [ ] แสดงยอดเหรียญที่มี (ถ้าพอ → เสนอให้จ่ายด้วยเหรียญ)
- [ ] ปุ่ม "ชำระเงิน" → เลือกวิธีชำระ (เหรียญ/บัตร/QR)
- [ ] ตะกร้าว่าง → แสดงข้อความ + link ไปหน้ารายการหนังสือ

### Checkout Flow
- [ ] เลือกวิธีชำระ + ยืนยัน
- [ ] Poll order status จน `paid` / `failed`
- [ ] สำเร็จ → clear cart + แสดง "ซื้อสำเร็จ X เล่ม" + link ไป library
- [ ] ล้มเหลว → ตะกร้าไม่ล้าง + error message

## Definition of Done

- [ ] เพิ่มหลายเล่ม → checkout ครั้งเดียว → ได้สิทธิ์ครบ
- [ ] เพิ่มรายการเช่า → error (CART_BUY_ONLY)
- [ ] จ่ายล้มเหลว → ตะกร้าคงอยู่
- [ ] `tsc --noEmit` และ `lint` ผ่าน
